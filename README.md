# Cricket Stats

A full-stack cricket statistics platform that ingests ball-by-ball and match-level data into a PostgreSQL database and serves it through a FastAPI REST API. The ingestion pipeline pulls data from public cricket data sources, normalises it, and stores it for fast querying, while the API layer exposes endpoints for player profiles, match summaries, batting and bowling averages, head-to-head records, and more — making it easy to build dashboards, analytical tools, or fan-facing websites on top of comprehensive cricket data.
